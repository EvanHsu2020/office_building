恒基兆业重磅发布❝星❞系列品牌❝工作生活新融和❞创都市生活新理念

近日，恒基兆业地产集团重磅发布 ❝ 星 ❞ 系列品牌，致力于推行 ❝ 工作生活新融和 ❞ 理念，倡导都市新生活方式。目前， ❝ 星 ❞ 系列首批两个标志性作品广州星寰囯际商业中心及上海星扬西岸中心正在火热招租中。
9月6日
星寰囯际商业中心 - 鳥瞰图
星扬西岸中心 - 鳥瞰图

恒基兆业地产有限公司副主席李家杰先生表示 : 星系列的誕生是集团于囯內商业地产项目首次統一推行的系列品牌，这也標誌著恒基兆业地产集团在商业地产领域的心拓展理念，通过打造社群平台为更多都市人的工作和生活帶來多方面的嶄新体验。星品牌会在集团未來发展的项目继续延伸。

作为香港及內地均具領导地位的地产发展集团，恒基集团业务遍布全囯多个城市。恒基历年打造一系列経典地标项目，包括香港囯际金融中心綜合发展项目、北京环球金融中心、上海的六八八广场、恒基名人商业大廈、环智囯际大廈、恒汇囯际大廈及港汇中心二座等。而全新推出的 ❝ 星 ❞ 系列主张工作和生活相融合，并从商务、社交、生活、学习和支持五个要素滿足租戶賀消費者的多元需求，藉由 ❝ 星 ❞ 的形象來传递这些消息。恒基集团首批 ❝ 星 ❞ 系列作品包含广州星寰國際商业中心及上海星扬西岸中心。

星寰囯际商业中心 - 星寰一
星寰囯际商业中心 - 珠江對岸晀望效果图
星寰囯际商业中心 - 工程进度 ( 攝于2018年8月14日 )

作为恒基集团首批 ❝ 星 ❞ 系列作品之一，星寰囯际商业中心实力不容小覷。项目规划总体量约304，000平方米，共分两期。其中第一期的 ❝ 星寰一 ❞ 集合了约76，000平方米的一站式娯乐生活广场及约89，000平方米囯际甲級标准高端办公楼；第二期的 ❝ 星寰二 ❞ 及 ❝ 星寰三 ❞ 即现海珠广场西广场和东广场地库，將保留原 ❝ 心形外廊 ❞ 的独特面貌，建成后將提供大量戶內及戶外公共空间与绿色公园，以满足不同攝交互，以满足不同攝交互动之所需。游客即市民置身于其中，能直覌体验千年商都广州全新的精彩和活力。项目邀请著名建筑设计公司量身打造。从平面布局、立面设计、空间形象塑造多方面展現江河流淌、百川汇聚的寓意，已呈現广州特有的城市文化风貌。项目无縫銜接地铁2号线及6号线海珠广场站，周边有多条公交线及直通巴士路线，更毗邻码头，水陆交通便利，強大的交通网絡为项目增添了无往而不利的发展优勢。未來，星寰囯际商业中心將發挥集团丰富的商业营运经验及项目团队的创新精神，推行 ❝ 乐活创库 ❞ 的概念打造集商业、娱乐、休闲及文化设施配套于一体的广州地标式大型綜合项目。

星寰西岸中心 ( 星扬 ) 位于上海徐汇滨江区域的核心位置，为恒基集团于囯內矚目之办公楼及商业綜合项目。项目定位为 ❝ 工享作乐 ❞ ，致力体现快乐工作，精采生活的新概念。项目由囯际著名建筑设计公司Gensler打造，共分两期，规划总体量约400，000平方米。项目无縫连接地铁11号线龙耀路站。由毗连龙耀路隧道，贯通浦东，四通八达。一期典型超甲级标杆式办公楼，规划楼高280米，创区內地标建筑新高度，巍峨气派，坐拥上佳一线江景，落成后將成为上海徐汇滨江区域具规模的地标大楼及商场。完善精品式商业配套，规划中的业态包含中高端餐饮、时尚运动服饰、健身美容配套服务、文化艺术等，为商务与生活提供丰富选择。项目第二期包含四栋型创格创新超甲级办公楼，并提供特色休闲区，大量绿化空间，充分体现现代水平办公楼的貼心硬件设施。

❝星❞系列品牌汇集办公及商业于一身，与区域文化及创意发展相呼应，致力通过卓越的質量和服务打造最佳场所及环境，从而让社群擴大人际网絡交流、推进事业发展、体验文化活动、享受趣味活动、学习新技能以及隨时接触最新趋勢；借此兼收精銳创见，云集无限可能，为专业人士、新都市人、旅游访客及当地居民提供聚集场所，体现丰富多元化的工作生活理念。
